#Amazing Space


link:https://aaronb22.github.io/Project_1/


technologies used:
  -HTML5
  -CSS
  -Javascript
  -APIs from NASA
  -Jquerry
  -Moment.js
  
In this app, the user starts at a homepage where the see the NASA picture of the day. They can select a datepicker and see past pictures. They also can go over to a info page about MArs, and see the current weather on Mars. Also the user can select a Mars rover and see an array of pictures from that rover. Also the user can pick a differnt sol to look at for the rover's imgs.
